# Changelog

All notable changes to Agent Deck will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial implementation of Agent Deck Native
- Daemon with PTY management and IPC server
- Tauri GUI with SolidJS frontend
- Session management (create, stop, delete, fork)
- Nested group organization
- xterm.js terminal emulator
- Claude status detection
- Settings modal with theme/font customization
- Keyboard shortcuts (Cmd+N, Cmd+,, Cmd+W)
- Toast notifications for connection state
- Auto-reconnection with exponential backoff
- Cross-platform support (macOS, Linux, Windows)
- GitHub Actions CI/CD workflows
